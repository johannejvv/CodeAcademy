CSS Reference Guide

border-* properties

Property Description & Syntax & Example
border It sets the values of border-width, border-style, and border-color
border: width style color
border: 2px double #32a1ce;
Can use any combination of values described below
border-width Sets the width of an element's border
border-width: top right bottom left
border-style: dashed groove none dotted;
Can use any non-negative legth (em or px) or a keyword (thing, medium, thick)
border-style Sets the line style for all four sides of an element's border
border-style: top right bottom left
border-style: dashed groove none dotted;
Can use any of predefined keyword values eg. dotted, dashed, solid
border-color Sets the color of an element's border
border-color: top right bottom left
border-color: red yellow green transparent;
Can use any combination of named color, hex values, rgba, or hsla
border-radius Rounds the corners of an element's outer border edge
border-radius: top-left top-right bottom-right bottom-left
border-radius: 10% 30% 50% 70%;
Units are length (px or em) or percentage values
